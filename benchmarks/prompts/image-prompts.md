# Clay benchmark — input image prompts

These generate the single-object input images for the TRELLIS-2 image→3D
benchmarks. Attach the **CLAY logo image** as a reference so the brand mark is
baked onto each object. Requirements for good 3D reconstruction: one centered
object, clean neutral studio background, soft even lighting, the whole object in
frame (no cropping), no overlaid text, no extra objects.

Save the results as `benchmarks/assets/<name>.png` matching `cases/default.json`.
